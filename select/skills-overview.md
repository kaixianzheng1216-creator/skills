# Skills 概述

| Skill | 核心功能 | 使用工具 |
| --- | --- | --- |
| `doc-coauthoring` | 协作撰写 PRD、RFC、技术方案、决策文档、提案；按上下文收集、分节打磨、读者测试推进。 |  |
| `docx` | 创建、读取、编辑 Word；处理修订、批注、表格、图片、目录、页眉页脚和格式转换。 | `defusedxml`, `lxml`, `docx-js`, `pandoc`, LibreOffice, `unpack.py`, `pack.py`, `validate.py` |
| `frontend-design` | 创建页面、组件、Web app、Dashboard；关注视觉设计、响应式布局和真实交互。 | HTML/CSS/JS, React, Vue, Tailwind, 图标库, Playwright/浏览器截图 |
| `internal-comms` | 编写 3P、周报、FAQ、Newsletter、领导汇报、项目更新、事故报告等内部沟通稿。 |  |
| `pdf` | 读取、提取、创建、合并、拆分、OCR、加密、加水印、填写 PDF 表单。 | `pypdf`, `pdfplumber`, `reportlab`, `pdf2image`, `pytesseract`, `pypdfium2`, `Pillow`, `qpdf`, Poppler |
| `pptx` | 创建、读取、编辑 PPT；支持模板适配、幻灯片增删改、内容提取、缩略图和视觉 QA。 | `markitdown[pptx]`, `Pillow`, `defusedxml`, `lxml`, `pptxgenjs`, LibreOffice, `pdftoppm` |
| `skill-creator` | 创建、修改、评测、优化和打包 Skill；支持 eval、baseline、grader、benchmark、review viewer。 | `PyYAML`, `json`, `pathlib`, `subprocess`, `quick_validate.py`, `package_skill.py`, `generate_review.py` |
| `xlsx` | 创建、读取、编辑、清洗 Excel/CSV/TSV；支持公式、格式、图表、财务模型和公式校验。 | `pandas`, `openpyxl`, LibreOffice, `recalc.py` |
